# Multiplication Table

## Description

Write a simple program to get a number from the user and create a multiplcation table for this number.

## Example

```text
Input - 5
Output - 5 * 1 = 5
Output - 5 * 2 = 10
Output - 5 * 3 = 15
Output - 5 * 4 = 20
Output - 5 * 5 = 25
Output - 5 * 6 = 30
Output - 5 * 7 = 35
Output - 5 * 8 = 40
Output - 5 * 9 = 45
Output - 5 * 10 = 50
```
