# Reverse Array

## Description

Get from the user a bunch of numbers, and after that, print them in reverse order.

## Example

```text
Input - 1
Input - 3
Input - 2
Input - x
Output - [2, 3, 1]
```

## More Challenging

- [ ] Print the negatives numbers;
- [ ] Return the some of all;
- [ ] Return the number of even numbers;
- [ ] Sort the array;
