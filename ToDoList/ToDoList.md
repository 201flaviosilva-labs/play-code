# To-Do List

## Description

Weel as the name says, let's create a to do list.

## Example

Menu:

```text
[1] -> Add
[2] -> Remove
[3] -> Print
[4] -> Exit
```

Add:

```text
Input - 1
Output - Write the task to add:
Input - Study programming
```

Remove:

```text
Input - 2
Output - Write the task to remove:
Input - Study programming
```

Print:

```text
Input - 3
Output - All the tasks:
Output - Task 1
Output - Task 2
```
